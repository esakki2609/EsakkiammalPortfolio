# Esakkiammal Portfolio

Professional portfolio for **Esakkiammal Palanisamy**, Senior Full Stack Developer. Built with Next.js and Tailwind CSS, branded with the olive logo color `#588F27`.

## Pages

- `/` — Home
- `/skills` — Skills
- `/experience` — Work experience
- `/projects` — Projects
- `/certifications` — Certifications & education
- `/contact` — Contact

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
